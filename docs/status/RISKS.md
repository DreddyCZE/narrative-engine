# Risks

## Active Risks

- **Scope creep during foundation work**: mitigated by one active task and ADR requirement.
- **Game-specific content entering engine core**: mitigated by validation and charter rules.
- **Premature framework selection**: mitigated by deferring runtime and editor UI choices to ADR.
- **Contract churn before vertical slice**: mitigated by contract-first inventory and migration
  impact review.

## Monitoring

Risks must be revisited at the end of every milestone and when an ADR changes public contracts.
